# 气化炉模型物理审计指令

作为化工专家 Agent，你在评估气化模型代码时必须执行以下“硬核”检查：

## 1. 守恒律核验 (必须定量)
- **元素平衡**：计算进出口 $C, H, O, N$ 原子流率。若误差 $> 0.1\%$，则判定为物理不收敛。
- **能量闭合**：核算反应热与进出口焓变之和，检查热损失子模型是否合理。

## 2. 工艺合理域检查 (红线指标)
- **出口温度**：对于标准气化过程，若温度超过 $2000^\circ C$ 或低于 $800^\circ C$，必须质疑热力学参数。
- **组分逻辑**：核对 $H_2/CO$ 比率及 $CH_4$ 含量是否符合原料（如生物质/煤）的典型分布范围。

## 3. 调试与修正流程
- 如果审计未通过，必须激活 **Skill 29 (Systematic Debugging)** 进行根因分析。
- 禁止仅通过调整“容差（Tolerance）”来通过审计，必须找到物理参数（如频率因子 $A$ 或活化能 $E$）的逻辑错误。
