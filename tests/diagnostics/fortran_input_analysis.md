# Fortran 工况输入条件分析报告

## 温度 < 1200°C 的工况检查

### 1. 入口水输入检查

**结论：无重复输入问题**
- 所有工况的 `SlurryConcentration=100%`（干粉进料）
- `fsteam` 为纯外加蒸汽，不包括煤浆水
- `xmois` 为煤水分，单独计算

### 2. Steam/Coal 比例合理性检查

#### Texaco 工况典型值范围
- **文献典型值**：Steam/Coal = 0.1 - 0.3（质量比）
- **Fortran 输入文件中的值**：

| 工况 | fsteam | 出口温度(°C) | 状态 |
|------|--------|--------------|------|
| texaco i-1 | 0.241 | 1283.7 | ✓ 正常 |
| texaco i-2 | 0.314 | 956.4 | ⚠️ 略高，温度偏低 |
| texaco i-5c | 0.429 | 1079.0 | ⚠️ **明显偏高**，温度偏低 |
| texaco i-10 | 0.276 | 1149.0 | ✓ 正常 |
| texaco exxon | 0.500 | 940.6 | ⚠️ **严重偏高**，温度过低 |

#### 问题分析

**texaco exxon** (fsteam=0.5):
- Steam/Coal = 0.5 远超 Texaco 典型范围（0.1-0.3）
- 出口温度仅 940.6°C，明显偏低
- **建议**：检查原始文献或 Fortran 输入文件是否有误

**texaco i-5c** (fsteam=0.429):
- Steam/Coal = 0.429 明显高于典型范围
- 出口温度 1079.0°C，偏低
- **建议**：核实该工况是否为特殊实验条件

**texaco i-2** (fsteam=0.314):
- Steam/Coal = 0.314 略高于典型范围上限
- 出口温度 956.4°C，偏低
- **建议**：可接受，但需确认是否为高蒸汽工况

#### Slurry 工况
- `slurry western`: fsteam=0.52（非 Texaco，可接受）
- `slurry eastern`: fsteam=0.79（非 Texaco，可接受）

### 3. 建议修正方案

#### 方案 A：保持 Fortran 原始值（如果文献确认）
- 如果 Fortran 输入文件来自可靠文献，保持原值
- 但需在文档中标注这些为"高蒸汽工况"

#### 方案 B：修正明显异常值
- `texaco exxon`: fsteam 从 0.5 降至 0.25-0.3
- `texaco i-5c`: fsteam 从 0.429 降至 0.3-0.35

#### 方案 C：检查 Fortran 输入文件单位
- 确认 `fsteam` 是否为质量比（kg/kg）而非其他单位
- 检查是否有其他隐含的水输入

### 4. 验证建议

1. **查阅原始文献**：确认这些 Texaco 工况的 fsteam 值是否与文献一致
2. **对比 Fortran 输出**：如果可能，运行原始 Fortran 代码，查看其预测温度
3. **敏感性分析**：测试 fsteam 对温度的影响，确定合理范围
