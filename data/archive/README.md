# 归档说明

本项目下的验证案例已于 2026-03-28 完成标准化组织。

## 归档文件
- `validation_cases.json`: 旧的索引文件。
- `validation_cases_merged.json`: 由 `merge_validation_cases.py` 生成的中转合并文件。
- `validation_cases_fortran.json`: 原始 Fortran 迁移案例（已弃用）。
- `validation_cases_OriginalPaper.json`: 1979 论文原始数据。
- `validation_cases_new.json`: 新增工业算例。
- `validation_cases_industrial.json`: 工业规模数据。
- `validation_cases_pilot.json`: 小试规模数据。

## 当前主力文件
- `data/validation_cases_final.json`: 经过标准化、分类和去重后的最终版本。
