## 10. 关键假设差异汇总

| 假设 | Fortran | Python | 影响 |
|------|---------|--------|------|
| 扩散系数 | 各反应经验公式，T^0.75 | Fuller 统一公式，T^1.75 | 传质阻力不同，影响异相速率 |
| WGS rat 因子 | 启用，高温压低 | 默认关闭 | Python 高温 WGS 可能偏快 |
| MSR 温度阈值 | T≤1000K 关断 | 无 | Python 低温可能仍有 MSR |
| C+O2 粒径修正 | dp 5–100 mm 线性插值 φ | 无 | 大颗粒时 CO/CO2 比可能不同 |
| 挥发分燃烧优先级 | tar > CH4 > CO > H2 | CH4 > CO > H2 | O2 不足时分配不同 |
| 热损失比例 | 固定 30% | 可调 1.5%–2% | 出口温度敏感 |
| C+O2 速率因子 | 无 | 可调 0.2–0.3 | 燃烧区长度、温度分布不同 |

---

## 11. 建议的修正方向（针对鲁南温度偏低）

基于上述差异，可优先尝试：

1. **扩散系数**：在 Python 中实现 Fortran 式经验扩散公式，或标定等效系数。
2. **WGS rat 因子**：在高温工况下启用 `rat=exp(-8.91+5553/T)`，观察 CO/H2 与温度。
3. **热损失**：保持 HeatLossPercent 在 1.5%–2% 的合理范围。
4. **CharCombustionRateFactor**：继续微调（如 0.15–0.25），延长燃烧区。
5. **O2 阈值**：尝试 0.02–0.03 atm，观察燃烧区扩展效果。
