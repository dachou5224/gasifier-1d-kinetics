## 2. 现象分析

### 2.1 温度变化

- **修改前**：出口温度系统性偏低（约 136–271°C）
- **修改后**：出口温度系统性偏高（约 417–791°C）
- **结论**：P0/P1 修改使净放热显著增加，温度由“偏低”转为“偏高”

### 2.2 气体组成

- **CO 偏低、CO2 偏高**：与修改前一致，说明 WGS 正向或氧化仍偏强
- **H2 偏低**：与 CO 偏低、CO2 偏高一致（WGS 正向：CO→CO2，H2O→H2，但若氧化主导则 H2 被消耗）

### 2.3 可能原因

1. **P0-1 删除 Step 9**：恢复完整挥发分，约 +28 MW 燃烧热 → 温度显著升高
2. **P0-2 固相焓简化**：去掉 Hf_char 经验公式，焓基准变化可能影响能量平衡
3. **P1-3 异相活化能**：E 增大（如 C+O2 149.4 kJ/mol）→ 异相速率变慢，但若挥发分燃烧主导，净效应可能仍为升温
4. **P1-4 均相动力学**：MSR E 从 249 降至 125.5 kJ/mol → 甲烷重整加快（吸热）→ 应降温，但当前温度仍偏高，说明放热项占优

## 3. 温度偏高原因分析（三步检查结果）

### 3.1 分步回滚 P0-1、P0-2（scripts/rollback_p0_analysis.py）

| 配置 | T_out (°C) | ΔT vs 当前 |
|------|------------|------------|
| 当前（P0-1/P0-2 均已修改） | 1929.8 | 0 |
| 回滚 P0-1（恢复 Step 9） | 1929.8 | **0** |
| 回滚 P0-2（恢复 Hf_char） | 2018.5 | **+88.7** |
| 同时回滚 P0-1 + P0-2 | 2018.5 | +88.7 |

**结论**：
- **P0-2 固相焓简化**：当前修改（去掉 Hf_char、统一用 Hf_coal）**降低了**温度约 89 °C，是降温项，非升温原因
- **P0-1 删除 Step 9**：回滚后温度不变，说明 Step 9 截断条件在当前工况下**未触发**（或挥发分已接近目标），P0-1 不是当前温度偏高的直接原因
- 温度偏高的主要来源应在 **P1 动力学修改** 或 **挥发分产率本身**（元素守恒产率已足够大）

### 3.2 Cell 0 能量审计（audit_cell0_energy.py）

| 工况 | Cell 0 T_out | Q_rxn_total | energy_src (evap) | ΔH |
|------|--------------|-------------|-------------------|-----|
| Paper_Case_6 | 2018.5 °C | 289 MW | -115.8 MW | -116 MW |
| LuNan_Texaco | 2024.6 °C | 109 MW | -40.7 MW | -40.8 MW |

- 能量平衡一致：ΔH = energy_src - Q_loss
- 蒸发热沉（energy_src）占反应放热约 40%，与浆液/水分蒸发量相符

### 3.3 WGS 催化因子核对

| 来源 | 催化因子 f | 结论 |
|------|------------|------|
| Fortran wgshift L1267 | `f = 0.2` | - |
| Python kinetics_service.py L223 | `WGS_CATALYTIC_FACTOR = 0.2` | **一致** |

### 3.4 温度偏高原因归纳

1. **P0 修改非主因**：P0-2 实际降温，P0-1 回滚无变化
2. **挥发分产率**：元素守恒产率可能偏高，导致燃烧放热过大
3. **P1 异相动力学**：活化能修正后 C+O2 等速率变化，需与 Fortran 逐反应核对
4. **均相燃烧**：CO/H2/CH4 氧化速率、phi 机制需与 Fortran 对比
