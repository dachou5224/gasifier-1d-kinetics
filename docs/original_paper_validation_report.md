## 4. 评估总结

### 4.1 温度

- **Texaco_I-1 严重偏低**：804°C vs 1370°C 实验，与 `docs/temperature_diagnosis.md` 中现象一致
- **Texaco_I-2 与 Texaco_Exxon 明显改善**：1149–1226°C，更接近实验范围
- **高蒸汽比工况（I-5C、I-10）偏低**：751–764°C，可能与 WGS 吸热、气化区冷却有关

### 4.2 组成

- **H2**：普遍接近实验，偏差多在 1–2 pp 以内
- **CO**：部分偏高（I-2）、部分偏低（I-1），规律性不强
- **CO2**：Texaco_I-1 偏高（7.27 vs 2.95），说明 WGS 平衡或碳转化路径与实验有差异
- **CH4**：模型几乎为 0，与实验 0.1–0.2% 存在差距（MSR 修正后仍偏弱或消耗过快）

### 4.3 可能原因

1. **起燃区温度初值**：Cell 0 高温猜测与 Fortran 不完全一致
2. **WGS 判据**：Ts≤1000K 关闭 WGS 的阈值及适用范围
3. **颗粒温度**：是否启用 RK-Gill 燃烧区颗粒温度、对气化区 WGS 的影响
4. **热损失**：3% 热损失是否与文献一致、轴向分布假设
5. **MSR 活化能**：已修正为 125.6 kJ/mol，对 CH4 转化、温度分布的影响需进一步验证

### 4.4 建议

1. 对 Texaco_I-1 做轴向温度与反应速率剖面诊断
2. 对比 Fortran 与 Python 在 Texaco_I-1 与 Texaco_Exxon 的轴向分布
3. 评估启用 `USE_RK_GILL_COMBUSTION` 对出口温度与组成的影响
